## 0.18.0 &mdash; 2026-03-14

The **OKLCH + Framer Motion** release. Three major system-wide migrations:

- **OKLCH Color Tokens** &mdash; All color primitives migrated from hex shade numbers (50&ndash;950) to OKLCH functional steps (1&ndash;12). Dark mode is now algorithmically derived with perceptually uniform lightness/chroma curves.
- **Framer Motion** &mdash; All interactive animations migrated from CSS keyframes to physics-based springs. 37 components migrated. New `MotionProvider`, spring/tween presets, and 6 motion primitives.
- **Toast API rewrite** &mdash; Imperative `toast()` function replacing the old component-based API.

### Breaking Changes

- **Color tokens**: `--pink-50` through `--pink-950` &rarr; `--pink-1` through `--pink-12` (OKLCH). Same for all 8 scales.
- **Semantic tokens**: `--color-interactive` &rarr; `--color-accent-9`, `--color-text-secondary` &rarr; `--color-surface-fg-muted`, and many more. Old names preserved as aliases.
- **Transitions removed**: `Fade`, `Collapse`, `Grow`, `Slide` from `./ui/transitions` &rarr; use `MotionFade`, `MotionCollapse`, `MotionSlide` from `./motion/primitives`
- **18 CSS keyframes removed** from the Tailwind preset (replaced by Framer Motion)
- **`useReducedMotion` hook removed** &mdash; use `MotionProvider` instead

### Added

- **Framer Motion system**: `MotionProvider`, `springs` presets (snappy, smooth, bouncy, gentle), `tweens` presets (fade, colorShift), 6 motion primitives
- **Spinner v2**: State machine (spinning/success/error), filled/bare variants, Framer Motion arc animation
- **Button `onClickAsync`**: Promise-driven loading &rarr; success/error states
- **OKLCH scale generator**: `generateScale()` utility for creating 12-step palettes
- **61 new Props type exports** added to barrel files
- **New export paths**: `./motion`, `./motion/primitives`

### Fixed

- Switch visible border on unchecked state
- Badge accent color variants (step corrections)
- Link color tokens
- Toast accent bar colors
- Button async feedback colors
- SegmentedControl and Stepper token references
