# Iconography

Shilp Sutra uses [**Tabler Icons**](https://tabler.io/icons) (`@tabler/icons-react`) -- 5,000+ open-source MIT-licensed icons, all stroke-based on a consistent 24x24 grid.

---

## Sizing standard

Always use **token-based sizing** for icons. This ensures consistency and allows global resizing via design tokens.

| Token class | Size | Usage |
|---|---|---|
| `h-[var(--icon-sm)] w-[var(--icon-sm)]` | 16px | Small UI icons, dismiss buttons, inline indicators |
| `h-[var(--icon-md)] w-[var(--icon-md)]` | 20px | Default — most icons in buttons, nav, list items |
| `h-[var(--icon-lg)] w-[var(--icon-lg)]` | 24px | Prominent icons, empty state illustrations |
| `h-[var(--icon-xl)] w-[var(--icon-xl)]` | 32px | Hero icons, large illustrations |

## Usage

```tsx
import { IconPlus, IconSearch } from '@tabler/icons-react'

// Token-based sizing (recommended)
<IconPlus className="h-[var(--icon-sm)] w-[var(--icon-sm)]" />

// Default (20px) icon
<IconSearch className="h-[var(--icon-md)] w-[var(--icon-md)]" />

// Custom stroke weight (default is 2)
<IconSettings className="h-[var(--icon-md)] w-[var(--icon-md)]" stroke={1.5} />

// Inherits text color
<IconHeart className="h-[var(--icon-sm)] w-[var(--icon-sm)] text-[var(--color-text-error)]" />

// Using the size prop (useful in stories or one-off sizes)
<IconSearch size={18} />
```

## Typing icon props

When a component accepts an icon as a prop, use the `Icon` type:

```tsx
import type { Icon as TablerIcon } from '@tabler/icons-react'

interface CommandItem {
  label: string
  icon?: TablerIcon
}
```

## Naming convention

All Tabler icons follow the pattern `Icon` + PascalCase:

| Pattern | Example |
|---------|---------|
| Object | `IconCalendar`, `IconFile`, `IconUser` |
| Action | `IconPlus`, `IconTrash`, `IconDownload` |
| Direction | `IconChevronRight`, `IconArrowUp` |
| State | `IconEye` / `IconEyeOff`, `IconMoon` / `IconSun` |
