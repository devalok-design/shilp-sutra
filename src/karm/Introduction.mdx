# Karm Domain Module

Karm is a project management application built with Shilp Sutra. This module provides **43 domain-specific components** organized into sub-modules covering kanban boards, task management, AI chat, dashboards, client portals, specialized buttons, and admin panels.

---

## Board (3 components)

Drag-and-drop kanban board for visual task management.

| Component | Description |
|-----------|-------------|
| **KanbanBoard** | Full kanban board with multiple columns and drag-and-drop |
| **BoardColumn** | Single board column with header, task list, and add-task action |
| **TaskCard** | Compact task card with title, assignee, priority, and status |

---

## Tasks (7 components)

Task detail views and tab panels for deep task inspection.

| Component | Description |
|-----------|-------------|
| **TaskDetailPanel** | Full task detail panel with header, properties, and tabbed content |
| **TaskProperties** | Task metadata display (assignee, status, priority, dates, tags) |
| **ActivityTab** | Activity log timeline for a task |
| **ConversationTab** | Threaded conversation view within a task |
| **FilesTab** | File attachments list with upload and preview |
| **ReviewTab** | Code review and approval tracking |
| **SubtasksTab** | Nested subtask list with progress |

---

## Chat (5 components)

AI-powered chat interface for project conversations.

| Component | Description |
|-----------|-------------|
| **ChatPanel** | Full chat panel container with message list and input |
| **ChatInput** | Message input with send button and formatting options |
| **MessageList** | Scrollable message list with sender avatars and timestamps |
| **ConversationList** | Sidebar list of chat conversations |
| **StreamingText** | Animated streaming text display for AI responses |

---

## Usage

Karm components are imported from the `karm` barrel:

```tsx
import { KanbanBoard, TaskCard } from '@devalok/shilp-sutra/karm'
import { ChatPanel, ChatInput } from '@devalok/shilp-sutra/karm'
import { AdminDashboard } from '@devalok/shilp-sutra/karm'
```
