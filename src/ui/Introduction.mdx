# UI Primitives

Shilp Sutra includes **52 low-level UI primitives** built on top of Radix UI. These are the foundational building blocks used by all higher-level components in the system.

Every primitive:

- Uses **semantic CSS tokens** for consistent theming
- Supports **dark mode** out of the box
- Uses **forwardRef** for composability
- Follows WAI-ARIA patterns via Radix UI

---

## Core

| Component | Description |
|-----------|-------------|
| Button | Primary action trigger with multiple variants and sizes |
| Input | Text input field with optional icons and states |
| Label | Accessible form label |
| Separator | Visual divider (horizontal or vertical) |
| VisuallyHidden | Accessible hidden content for screen readers |
